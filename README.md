# ひらがな計算機 (Hiragana Calculator)

ひらがなを使って数値を入力する計算機です。

GitHub Pages: `https://petipetipetti.github.io/-Hiragana-Calculator/`

## Run Locally

**Prerequisites:** Node.js


1. Install dependencies: `npm ci`
2. Start the development server: `npm run dev`

## Deploy to GitHub Pages

The GitHub Actions workflow builds and deploys the site whenever changes are pushed to `main`.
In the repository settings, set **Pages > Build and deployment > Source** to **GitHub Actions** once.
